One Where my Mac's Command+V Stopped Working

  24 January, 2018

  Ajay Sharma

 MacOS, OS, OSX

It was Sunday of December 2017. I was doing my winter Big Data Hadoop Internship at Xavient Information System, Noida. In the sunday morning when i start working i found that my Command+V (Paste) is not working. It is horrible, isn’t it? We copy and paste all the time. If paste is not working then you have to type all the thing that you can copy that is not less than a bad dream. I usually don’t shut down my pc for 2 to 3 days; I always put it on Sleep mode. I thought this problem is happening because of this. I restart my mac but problem was still there.

How I make it Work?

You can’t leave without copy+paste so i have to fix this. I started searching on internet. I got some solutions.

Approach 1:
First I tried this that i read on Apple’s community forum -

  1. Shut down the computer.
  2. Unplug the computer's power cord.
  3. Wait fifteen seconds.
  4. Attach the computer's power cord.
  5. Wait five seconds, then press the power button to turn on the computer.

This isn’t work. Problem was still there :(

Approach 2:
I tried to stop running pboard daemon and then start running again -

$ launchctl stop com.apple.pboard
$ launchctl start com.apple.pboard

this also doesn’t work.

Approach 3:
Somewhere I read this solution -

  1. From the Apple menu, select System Preferences.
  2. Select Keyboard.
  3. Click the Keyboard tab.
  4. Click the Modifier Keys... button. The window shown below appears.
  5. From the Command Key menu, select Control. This lets Mac OS X know that you'll be using the control key as the primary modifier.
  6. Click OK.
  7. Close System Preferences.

This solution also doesn’t work. So I shut down my mac and open it next day and I found that my Command+V is working properly. Peace!
If your paste command stop working then try above approaches, may be one of them fix your problem.
Thank you for reading :)
